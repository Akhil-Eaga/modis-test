Technical Details of this website:

  1. This website is served using the Apache Web Server
  2. This website is hosted on an EC2 instance on AWS
  3. The EC2 instances runs behind an Autoscaling Group
  4. A public facing Elastic Load Balancer is used to direct the incoming traffic
  5. Terraform is used to provision all these AWS resources
